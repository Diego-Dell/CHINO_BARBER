# CHINO_BARBER
💈 CHINO BARBER – Barber School Management System
Sistema de gestión integral para escuelas de barbería, desarrollado con Electron + Node.js + SQLite, diseñado para funcionar 100% offline, con control académico, financiero e inventario en un solo lugar.

Tecnologías usadas
•	Electron
•	Node.js
•	Express
•	SQLite
•	electron-builder
•	electron-updater
•	HTML / CSS / JavaScript
